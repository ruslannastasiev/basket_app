* Ruby version
  `ruby 2.3.0`
* Installation
  `git clone git@github.com:ruslannastasiev/basket_app.git`
  `cd basket_app`
  'bundle install'
  `bundle exec rails s`  

* Database creation
  `rake db:setup`
  ------ or ------
  `rake db:create`
  `rake db:migrate`
  `rake db:seed`

  Also you can use database from repo

* How to run the test suite
  `rspec spec`
